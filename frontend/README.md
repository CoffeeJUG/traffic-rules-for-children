## Steps to build
 - install git
 - clone git project
 - install NodeJS
 - build project
 - run application

### Install Node JS
Refer to https://nodejs.org/en/ to install nodejs
Make sure You have 
```sh
$ node -v
 v 12.18.4
$ npm -version
6.14.6
```
(minor version could be different)

### Build the project
```bash
npm -i
```

### Run the application
In order to run the application Type the following command

```bash
npm start
```

The Application should occurs on **http://localhost:3000**
